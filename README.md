## How to run?

1. Install dependencies

```
$npm install
```

2. Run

```
$npm run dev
```

3. Grab the local url and paste it in the browser

```
VITE v4.0.1  ready in 3782 ms

  ➜  Local:   http://127.0.0.1:5173/
  ➜  Network: use --host to expose
  ➜  press h to show help
```
